Projects

Tasky

React, Css

Tasky is a to-do list application built in React.js. Tasky allows the user to enter a task with a specified date and an optional description of the task. The user may add as many tasks as they would like and when they are finised with the task, they can delete the task. Tasky has a simple, functional UI. The input form for creating a task is a side bar allowing the input fields to be stacked vertically and efficiently use space. The tasks are then displayed in the remaining area in a grid like structure.

The Stoics

Html5, Css

The Stoics is a simple yet stylistic site made using pure HTML and CSS. The premise of the site is a club and is used to provide information on the club, bring in new members, and post events. The design has an elegant color pallete of dark reds mixed with tans and gold. The Merriweather serif font provides a sense of structure and reminds the viewer of the architecture of ancient Rome and Greece, while the Mullish sans-serif font provides a touch of modernity.
